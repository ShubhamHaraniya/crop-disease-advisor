# 🌾 Crop Disease Advisor

AI-powered plant disease diagnosis and treatment recommendations for Indian farmers.

## Features
- 🖼 **Drag-and-drop image upload** with instant preview
- 🦠 **EfficientNet-B4** vision model (38 disease classes, 14 crops)
- 🎚 **Dual severity mode** — Auto or Manual slider
- 💊 **Tabbed treatment plans** — Organic, Chemical, Preventive, Advisory
- 📄 **Professional PDF report** — downloadable clinical report
- 🌍 **Region & Season aware** — tailored for 5 Indian regions & 3 seasons

> Running in CPU mode — treatment plans powered by curated agricultural knowledge base.
